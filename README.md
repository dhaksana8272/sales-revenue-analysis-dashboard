# Sales & Revenue Analysis Dashboard

A Streamlit-based dashboard for analyzing sales and revenue data.

## Features

- CSV Import
- Excel Import
- SQLite Database Support
- KPI Tracking
- Revenue Trend Analysis
- Category & Region Analysis
- Business Insights
- Interactive Filters

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- SQLite

## Run Locally

pip install -r requirements.txt

streamlit run app.py
